This folder includes the implementations of Example 4.1 - 4.4 from the paper.

For details on defining a problem, refer to "ProblemDescriptionTemplate.hpp."

Compilation: 
            mkdir build && cd build
            cmake ..
            make

Run: After the compilation, four executable file will appear inside the build folder. Simply try
            ./basic
            ./rosenbrock
            ./hs071
            ./arm
